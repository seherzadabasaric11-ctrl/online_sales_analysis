#Online Sales Analysis

This project is a simple Python application for managing products and a shopping cart.
It demonstrates object-oriented programming concepts, Git version control,and basicproject structure.

 ##Features
 -Add and remove products
 -List available products 
 -Add products cart
 -Remove products from cart
 - Calculate total cart value

##Project structure 
-"product.py" -Product class
-"product_menager.py"- Product menagement logic
-"cart.py" - Sopping cart functionality
-"main.py" - Application entry point

##How to Run 
1.Clone the repository
2.Run the application: "" bash 
python main.py
